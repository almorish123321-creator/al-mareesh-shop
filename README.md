# 🏪 المريش شوب | Al-Mareesh Shop

متجر إلكتروني متكامل لملابس الأطفال والنسائية والأحذية

## ✨ المميزات

- 🛍️ واجهة متجر احترافية بالعربية (RTL)
- 📦 إدارة المنتجات والفئات
- 🛒 سلة تسوق متكاملة مع كوبونات خصم
- 💳 دعم طرق دفع متعددة (عند الاستلام، بطاقة، PayPal)
- 👥 نظام تسجيل ودخول المستخدمين
- 📊 لوحة تحكم إدارية شاملة
- 🔍 بحث وتصفية المنتجات
- 📱 تصميم متجاوب لجميع الأجهزة
- 🎨 تصميم أنيق بألوان مميزة (عنابي وذهبي)

## 🛠️ التقنيات

- **Next.js 16** مع App Router
- **TypeScript** 
- **Tailwind CSS 4** + **shadcn/ui**
- **Prisma ORM** مع SQLite
- **Zustand** لإدارة الحالة
- **Lucide React** للأيقونات

## 🚀 التشغيل

```bash
# تثبيت التبعيات
bun install

# إعداد قاعدة البيانات
bun run db:push

# تشغيل المشروع
bun run dev
```

src/

├── lib/

│   ├── store.ts      # Zustand Store
│   └── utils.ts      # أدوات مساعدة

    └── schema.prisma # مخطط قاعدة البيانات
